# McCTL

MConnect Control - control available mconnect devices

## Usage

Run `mcctl help` for usage instructions

## Dependencies

Compile-time:  
  - crystal
  - shards

Run-time:  
  - mconnect
  - dbus
  - gc
  - libevent

## Installation

`make && make install`  

Alternatively, use [this pacman PKGBUILD](https://github.com/coderobe/pkgbuilds/blob/master/mcctl-git/PKGBUILD)
